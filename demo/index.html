@arach/devbar

A beautiful, minimal development toolbar for React applications

Features

  • ✓ Multiple positioning modes (corners + pane)
  • ✓ Chrome DevTools-style bottom pane
  • ✓ Resizable height with drag handle
  • ✓ Light/Dark theme support
  • ✓ Tab-based interface
  • ✓ Zero dependencies (except React)

Installation

npm install @arach/devbar
